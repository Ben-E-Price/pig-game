Game Rules

  • Each player takes turns rolling the dice
  • The value of the dice is added to the players current score.
  • Using the hold button adds the current score to the players overall score, However using the hold also passes the game to the next player
  • If a player rolls a 1, The current score is lost and the next player takes their turn
  • The first player to reach an overall score of 100 wins.
